该项目是一个基于 Laravel 5.3 的一个小型商城，该商城由我和小伙伴开发的一个测试商城，主要娱乐，喜欢或者爱好开发商城的朋友可以参考玩玩，商城可能有些地方不够完善，但是是适合一个小型商城的使用。

## 开始

克隆项目

```
git clone https://github.com/iiDestiny/LaraMall.git
```

安装组件

```
composer install
```

配置数据库信息，还有七牛信息，创建数据库表

```
php artisan migrate
```
